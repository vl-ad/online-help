Searching for Content

Search is a function of the EdgeRover app that allows you to easily initiate a search in the app and search across all cataloged content.

For additional ease, begin typing into the search box to see suggestions based on previous searches.

Search examples:

  • Search by file name, such as Resume.
  • Search by file type, such as PDF.
  • Search photos by object name, such as Car.
  • Search photos by location, such as San Francisco or California.
  • Select from a list of previous search queries.
