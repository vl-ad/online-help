How do I update the EdgeRover app to the latest version?

To find the version of the application you are using:

  1. While in the EdgeRover app, click on Settings located in the left panel.
  2. Click on App Settings.
  3. To the right of App Version information, you will see the most current version of the software installed on your computer. If you are on the latest version, you will see You have the latest version of EdgeRover!.
  4. If you are not on the latest version of the software, you will see an Update now option. Selecting this option will download and install the latest version of the software on your computer.
